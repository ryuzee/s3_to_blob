# Copy S3 Objects to Azure Blob

```
Usage: copy_from_s3_to_blob [options]
    -a, --aws_access_key=[VALUE]     AWS Access Key
    -s, --aws_secret_key=[VALUE]     AWS Secret Access Key
    -r, --aws_region=VALUE           S3 Region
    -b, --aws_s3_bucket=VALUE        S3 Bucket Name
    -x, --aws_object_prefix=[VALUE]  S3 Object Prefix
    -z[VALUE],                       Azure Storage Account Name
        --azure_storage_account_name
    -k[VALUE],                       Azure Storage Access Key
        --azure_storage_access_key
    -c, --azure_container=VALUE      Azure Container Name
```
